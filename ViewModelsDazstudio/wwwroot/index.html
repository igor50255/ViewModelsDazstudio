Приложение для просмотра моделей программы DazStudio

  • При старте программы вам предлагается выбрать папку для размещения контента. При выборе создаются рабочие папки — по названиям будет понятно, что и куда размещать.
  • Основной единицей является папка, названная по имени модели. В ней уже может быть архив и другая информация.
  • Для создания превью в папке должна быть хотя бы одна картинка. Там будет создана папка "preview" и в неё добавится первая картинка из папки модели. После создания, превью можно заменить вручную, но оставьте имя файла "0.*". Если картинок нет — будет использована дефолтная картинка.
  • При клике на название модели, открывается папка этой модели в проводнике.
  • Ctrl + D — включает полноэкранный режим и выводит из него.
  • При изменении пути к папке с контентом, старая папка не удаляется. Также если по новому пути уже есть папка — ничего не удаляется.
  • Если при скролле тормозит отображение картинок — сделайте окно приложения меньшего размера (мышью), это часто ускоряет отрисовку.
